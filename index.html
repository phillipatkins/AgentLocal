Quick Start
Running in minutes

The setup wizard handles everything — dependencies, Ollama, WhatsApp linking and configuration.

1

Clone the repo

Grab the latest release from GitHub

2

Run the setup wizard

Installs all dependencies and walks you through config

3

Scan the QR code

Link your WhatsApp account — takes 10 seconds

4

Start the bot

Your AI is live and ready on WhatsApp

terminal
# Clone
git clone https://github.com/
  AgentLocal-hub/AgentLocal
cd AgentLocal

# Install & configure
npm run setup

# Launch
npm start

Commands
Control everything
from WhatsApp
search [query] Real-time web search
find magnet [query] Torrent search and download
youtube [url] Download audio or video
remind me [task] at [time] Set a smart reminder
remember [text] Save a memory note
stock alert [TICKER] [price] Set a stock price alert
use gpt / use grok Switch to GPT or Grok relay
/help Full command reference
